# AI Quiz Generator

-An AI-powered quiz web application that generates quizzes using the Gemini API.
-Add your Gemini API key in script.js before running the project.

## Features
- AI-generated quizzes
- Configurable number of questions
- Difficulty levels
- Timer
- Quiz history
- Voice answer support
- AI study insights
- PDF export
- Database integration (Flask + SQLite)

## Technologies Used
HTML, CSS, JavaScript  
Google Gemini API  
Python Flask  
SQLite Database

## How to Run

1. Install Flask
pip install flask

2. Run backend
python server.py

3. Open the frontend using Live Server.

4. Enter topic and generate quiz.
